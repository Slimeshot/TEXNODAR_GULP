Интерфейс Системы

Отчет — режим отображения данных в формате таблицы. Система отчетности разрабатывается индивидуально под требования заказчика. На основании полученных с судна данных система DOTS осуществляет необходимые вычисления и выдает результат в течении 30 секунд, в удобном формате. Количество отчетов - не ограничено.

Выбор типа отслеживания: мониторинг, мультитрек, трек и график, отчет
Мониторинг отображает, где судно находится в данный момент
Мульти-трек — сводные данные о маршруте судна за выбранный период времени
Трек и график — режим отображения данных, интересующих параметров с привязкой к маршруту судна
Отчет — режим отображения данных в формате таблицы. Система отчетности разрабатывается индивидуально под требования заказчика. На основании полученных с судна данных система DOTS осуществляет необходимые вычисления и выдает результат в течении 30 секунд, в удобном формате. Количество отчетов - не ограничено.
Для удобства управления данные упорядочены по базовым разделам: агрегаторы (содержаться данные по потребителям), состаяния (содержаться данные о состоянии работы потребителей), движения (показатели по характеристикам передвижения), все (совокупность полученных параметров) и кастомные поля, которые вносятся по желанию заказчика.
